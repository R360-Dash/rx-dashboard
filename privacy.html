Privacy Policy

Last updated: April 23, 2026

This Privacy Policy describes how the RacquetX Dashboard (the "Application"), operated by Racket X Expo LLC ("Company", "we", "us"), handles information accessed through connected third-party services. The Application is an internal reporting tool used solely by authorized Company personnel and stakeholders.

1. Information We Access

The Application connects to and displays read-only data from the following systems:

  • Intuit QuickBooks Online — Company financial records (Profit & Loss, Cash Flow, expense detail) for the Company's own QuickBooks account.
  • HubSpot CRM — Sponsorship and exhibitor deal pipeline data from the Company's HubSpot account.
  • Grip Events Platform — Attendee registration and ticket sales metrics for Company events.
  • Google Analytics (GA4) — Aggregated website traffic metrics.
  • LinkedIn Analytics — Aggregated social media performance metrics.

2. How We Use Information

Information accessed through the Application is used exclusively for internal Company reporting, forecasting, and business decision-making. The Application does not:

  • Store financial data from QuickBooks on its own servers;
  • Share, sell, or distribute any data to third parties;
  • Modify, create, or delete records in any connected service (access is read-only);
  • Process or display personally identifiable information of end customers.

3. Data Storage and Retention

The Application fetches data on demand from the connected services via secure API calls. Responses are cached temporarily (up to five minutes) in a Cloudflare Worker proxy to reduce API load, but no long-term copy of the data is retained outside the source systems.

4. Access Controls

Access to the Application is restricted to named users authenticated with unique credentials. Role-based access controls limit which sections of the Application each user can view.

5. Security

OAuth credentials, API tokens, and refresh tokens are stored as encrypted environment secrets within the Application's Cloudflare Worker infrastructure. The Application is served over HTTPS.

6. Third-Party Services

Use of each connected third-party service is governed by that service's own privacy policy:

  • Intuit QuickBooks Online: https://www.intuit.com/privacy/
  • HubSpot: https://legal.hubspot.com/privacy-policy
  • Google (GA4): https://policies.google.com/privacy

7. Changes

We may update this Privacy Policy from time to time. The "Last updated" date at the top of this page will reflect the date of the most recent revision.
